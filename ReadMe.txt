
You will write a Hacker News application that
interacts with a real world API. Among several interesting topics, it covers state management in
React, caching and interactions (sorting and searching).
-----How to run this app
npm install
npm start

-----How to create this app

-- React Projet create.
npm install create-react-project -g
create-react-project your-amazing-app(project name)

-- Runs the application in http://localhost:3000
npm start

-- Runs the tests
npm test

-- Builds the application for production
npm run build
